Símbolos e Emojis

Aqui eu irei mostrar muitos símbolos que você pode usar durante sua escrita em html, logo a seguir
< que é o sinal de menor que
> que é o sinal de maior que
  que é o espaço em branco
® que é o sinal de marca registrada
© que é o sinal de copyright
™ que é o trade mark, famoso TM
€ que é o simbolo do Euro
£ que é a Libra
¥ que é o Yen
∅ que é o sinal de vazio
∑ que é o sinal de soma
Δ que é o delta
← que é uma seta para a esquerda
→ que é uma seta para a direita
↓ que é uma seta para baixo
↑ que é uma seta para cima
